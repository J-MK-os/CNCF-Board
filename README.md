\# CNCF Team Project – 클라우드네이티브와운영기술 게시판



\## Project Overview

본 프로젝트는 \*\*로컬 장비를 활용해 클라우드 환경을 모사한 실제 서비스 개발과 운영을 경험\*\*을 목적으로,

게시판 웹서비스를 \*\*Kubernetes 기반으로 구성, 배포, 운영\*\*하는 것을 목표로 합니다.



프론트엔드 · 백엔드 · 데이터베이스를 각각 독립된 서비스로 구축하고,

컨테이너화 및 오케스트레이션, CI/CD, 모니터링, 서버리스 확장 기능을 적용했습니다.



\## Project Subject

프로젝트 목표

프론트엔드와 백엔드가 분리된 게시판 서비스 개발, 각각을 독립된 서비스로 구성하고, 서로 다른 컨테이너 및 Kubernetes Deployment로 배포

Git 기반 CI/CD 파이프라인 구성

모니터링 기능 구현

서버리스 함수 (AWS Lambda 등) 를 활용한 추가 기능 구현
